# v1.4.0

Release date: 2026-01-06

## Highlights

- Added the Kafka proxy deployment and demo wiring.
- Rolled out broker StatefulSets with a smoother publish flow.
- Expanded Kafka API compatibility (Create/DeleteTopics v0-2, ListOffsets v0-4).
- Hardened platform ops: etcd restore isolation/recovery, operator snapshot e2e stability, and expanded metrics/Helm wiring.
- Added the stateless Iceberg table processor and templates.
- Introduced KafScale processors for ephemeral compute in streaming pipelines.
- Added a benchmark suite.

## Known limitations

- TLS/SASL authentication is planned for v1.5+ (see roadmap).

## Bug fixes

- #47: Iceberg demo stability fixes and output improvements.
- #44: etcd auto-restore race can overwrite data; fixed restore isolation/recovery.
- #43: consume error on unstable etcd.
- #42: HPA broker scale-out advertised metadata fixes.
- #36: Apache Flink source failure fixes plus ListOffsets v0-4 support.

## Security fixes

- No known runtime vulnerabilities were fixed in this release.
